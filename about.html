About this project

This application is meant to allow you to visualize graphs and make searches over them. To do that, first you must load a graph. There are 3 ways to do this:

  • Loading your own graph on the top option from the left panel. The file must be in JSON or GEXF format. If it's a JSON file, it should include at least an array called "links". Each element from the collection must have a numerical source and target, and an optional weight. Once you load a valid graph, it will be saved in the server and it will send you back a hash to load the graph in future executions.
  • Selecting an example graph from the server.
  • Introducing the hash obtained by the first method.

Once your graph was loaded, you should be able to visualize it if everything went okay. After this, you can do graph-searches after selecting the initial and goal nodes by clicking on them and then marking the right checkboxes on the right. You can select the algorithm to do the search and the way to do it (directly or step by step). You can also change the distribution of the nodes by selecting and applying a layout. You can also change the visualization manually by dragging the nodes or the whole graphic, and zooming with the lower lens.

This project was created by Miguel López Becoña. Its graph-search component is Hipster, which was originally created and is currently maintained by:

Rodriguez-Mier, P., Gonzalez-Sieira, A., Mucientes, M., Lama, M. & Bugarin, A. (2014). Hipster: An Open Source Java Library for Heuristic Search. 9th Iberian Conference on Information Systems and Technologies (CISTI).
